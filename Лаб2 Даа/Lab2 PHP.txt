# ✅ Лабораторная работа №2

**Установка и первая программа на PHP**

---

## Шаг 1. Установка PHP (кратко)

1. Скачать PHP:
   [https://www.php.net/downloads](https://www.php.net/downloads)

2. Распаковать, например:

```
C:\Program Files\php
```

3. Добавить в PATH:

```
Win + R → sysdm.cpl
Дополнительно → Переменные среды → Path → Добавить:
C:\Program Files\php
```

4. Проверить установку:

```bash
php -v
```

Если отображается версия PHP — установка выполнена успешно.

---

## Шаг 2 (альтернатива). Установка через XAMPP

1. Скачать XAMPP:
   [https://www.apachefriends.org](https://www.apachefriends.org)

2. Установить компоненты:

* Apache
* PHP
* phpMyAdmin

3. Запустить **XAMPP Control Panel** → Start Apache.

4. Проверить:

```
http://localhost
```

---

---

## Шаг 5. Переменные и вывод

### Полный готовый код `index.php`

```php
<?php

declare(strict_types=1);

// Первая программа
echo "Привет, мир!<br /><br />";

// echo и print
echo "Hello, World with echo!<br />";
print "Hello, World with print!<br /><br />";

// Переменные
$days = 288;
$message = "Все возвращаются на работу!";

// Конкатенация
echo "Количество дней: " . $days . "<br />";
echo "Сообщение: " . $message . "<br /><br />";

// Двойные кавычки
echo "Количество дней: $days<br />";
echo "Сообщение: $message<br />";
```

# ✅ Контрольные вопросы

---

## 2. Как проверить, что PHP установлен?

В командной строке выполнить:

```bash
php -v
```

Если отображается версия PHP — всё работает.

---

## 3. Чем отличается echo от print?

| echo                           | print               |
| ------------------------------ | ------------------- |
| быстрее                        | немного медленнее   |
| может выводить несколько строк | выводит одну строку |
| не возвращает значение         | возвращает 1        |
| чаще используется              | используется реже   |

Пример:

```php
echo "A", "B";
print "A";
```
